# **App Name**: Zepmeds: Online Medicine Delivery

## Core Features:

- Login with 3D Elements and Motion Graphics: Cool landing page for login/signup page with 3D animated graphics on app start.
- Profile Management: Users can manage their profile information including name, phone number, address, order history, and other details.
- Medicine Search and Ordering: Enables users to search for medicines, view details, add to cart, and place orders. Include delivery card for ordering and other prescription uploads.
- Prescription Upload: Functionality for users to upload prescriptions via camera or gallery, for pharmacist review.
- AI-Powered Symptom Checker: AI chatbot to help users self-diagnose, suggests medicines and connects users to online consultation.
- Order Tracking & Delivery ETA: In Cart page display real-time tracking with estimated time of arrival of medicine deliveries and other potential savings tips.
- Payment Options: Integration of cash, card, UPI, and 'buy now pay later' payment options for transactions.

## Style Guidelines:

- Primary color: Dark Blue (#2f32bd) reminiscent of Farmako's theme for a professional and reliable look.
- Background color: Desaturated dark blue (#252899) to maintain the dark theme while reducing eye strain.
- Accent color: Purple (#7B68EE), which is analogous to dark blue and will draw users attention for CTAs and highlights.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text, offering a modern and readable feel.
- Use clear and modern icons to represent different medicine categories and features.
- Implement a clean and easily accessible UI layout, with intuitive navigation for categories, search, and user profile.
- Subtle transitions and background animations to enhance user experience and make the app visually appealing.
